6. Module quickff.refdata – Ab initio reference data¶

class quickff.refdata.ReferenceData(coords=None, grad=None, hess=None)[source]¶

A class to store the ab initio reference data from which the force field will be derived.

Optional Arguments

coords
A (N,3) numpy array containing the cartesian coordinates of all atoms in equilibrium.
grad
A (N,3) numpy array containing the cartesian gradient of all atoms in equilibrium. Ideally, all elements of grad are zero.
hess
A (N,3,N,3) numpy array containing the cartesian Hessian in equilibrium.
update(coords=None, grad=None, hess=None)[source]¶

Method to update one or more of the attributes. The meaning of the optional arguments is the same as with the constructor quickff.refdata.ReferenceData()
