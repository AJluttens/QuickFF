10. Module quickff.evaluators – Evaluators of observables on perturbation trajectories¶

quickff.evaluators.eval_ic(ic)[source]¶

Return an evaluator for the value of the given internal coordinate, i.e. return a method that calculates the value of the given ic for a certain atomic configuration.

Arguments

ic
an instance of the class quickff.ic.IC
quickff.evaluators.eval_energy(part_name)[source]¶

Return an evaluator for the value of the part_name energy, i.e. return a method that calculates the part_name energy for a certain atomic configuration.

Arguments

part_name
a string defining an attribute of a model instance that is needed to evaluate the correct part of the energy.
