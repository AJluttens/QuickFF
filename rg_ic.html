9. Module quickff.ic – Internal coordinate definitions¶

class quickff.ic.IC(name, indexes, icf, qunit='au', kunit='kjmol/au**2')[source]¶

A class functioning as a wrapper around the MolMod ic functions. This class should be able to calculate the value of an internal coordinate and its first and second order derivatives toward the cartesian coordinates.

Arguments

name
a string defining a unique name for the ic
indexes
a list of indices refering to the relevant atoms of the system
icf
a method from the molmod.ic module to calculate the value, gradient and hessian of the ic

Optional Arguments

qunit
a string describing the conversion of the unit of the ic value. More info regarding possible strings can be found in the MolMod documentation.
kunit
a string describing the conversion of the unit of the force constant. More info regarding possible strings can be found in the MolMod documentation.
grad(coords)[source]¶

Calculate the first order derivative of the internal coordinate

Arguments

coords
a (N,3) numpy array containing the cartesian coordinates of all atoms.
hess(coords)[source]¶

Calculate the second order derivative of the internal coordinate

Arguments

coords
a (N,3) numpy array containing the cartesian coordinates of all atoms.
value(coords)[source]¶

Calculate the value of the internal coordinate

Arguments

coords
a (N,3) numpy array containing the cartesian coordinates of all atoms.
