11. Module quickff.fftable – Table to store FF parameters¶

class quickff.fftable.DataArray(data=None, unit='au')[source]¶

A class to store data representing measurements (or calculations) of the same observable. Some basic statistics will be done automatically and can be accessed at all time.

class quickff.fftable.FFTable[source]¶

A class to read, store and dump Force Field parameters. The parameters of the force field term for an ic of kind icname can be accessed as follows:

k, q0 = fftable[icname]

The statistics of a parameter (e.q. the force constant of the term related to icname) can be accessed as follows:

k_mean = fftable.pars[icname][‘k’].mean k_std = fftable.pars[icname][‘k’].std k_num = fftable.pars[icname][‘k’].num
